# Devoryn Dashboard - Fullstack Modern Dashboard

Một dự án Dashboard quản trị hiện đại được xây dựng với mục tiêu thực chiến và áp dụng các tiêu chuẩn công nghiệp.

## 🚀 Công nghệ sử dụng

### Frontend
- **React.js** (Vite)
- **Vanilla CSS** (Futuristic Glassmorphism Design)
- **Lucide React** (Icons)
- **React Router Dom v7**

### Backend (Đang phát triển)
- **Node.js** & **Express**
- **TypeScript**
- **MongoDB** (Mongoose)
- **Docker** (Containerization)
- **Zod** (Validation)
- **JWT** (Authentication)

## 📦 Cấu trúc dự án
- `/frontend`: Mã nguồn phía người dùng (React).
- `/backend`: Mã nguồn server (Node.js + TypeScript).

## 🛠 Cách chạy dự án (Local)

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`

### Backend
1. `cd backend`
2. `npm install`
3. `npm run dev`

---
*Dự án đang trong quá trình phát triển 28 ngày để hoàn thiện Backend.*
